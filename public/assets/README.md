# Technology Icons

Place your technology logo PNG files in this directory with the following names:

- `react.png` - React logo
- `typescript.png` - TypeScript logo
- `nodejs.png` - Node.js logo
- `python.png` - Python logo
- `postgresql.png` - PostgreSQL logo
- `aws.png` - AWS logo
- `nextjs.png` - Next.js logo
- `graphql.png` - GraphQL logo

**Recommended specifications:**
- Format: PNG with transparent background
- Size: 64x64 pixels or larger (will be scaled down to 48x48)
- Aspect ratio: Square (1:1)

You can download these logos from:
- Official technology websites
- Simple Icons: https://simpleicons.org/
- Devicon: https://devicon.dev/
